A taxa de acerto do modelo SVM foi de 80%, sendo 20% maior do que o modelo da Árvore de decisão (Repositório no Github), que obteve 60% como taxa de acerto.


Isso ocorre devido a algumas características do SVM, como :

  • Em caso de outlier a SVM busca a melhor forma possível de classificação e, se necessário, desconsidera o outlier;
  • Não funciona bem em conjuntos de dados muito grandes;
  • Se as classes estiverem muito sobrepostas deve-se utilizar apenas evidências independentes (devido ao fato de não ser muito bom com dados com muitos ruídos).


Já a Árvore de decisão necessita de uma quantidade maior de dados para aumentar sua precisão. Então quando há um outlier, um dado que se afasta muito dos padrões do conjunto de dados, ela o classifica como um dos padrões, diferente do SVM.
